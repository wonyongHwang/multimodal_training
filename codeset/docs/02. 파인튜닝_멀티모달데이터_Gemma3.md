## 주요 하이퍼파라미터 (.env 제어)

| `.env` 키 | 기본값 | 설명 |
|-----------|--------|------|
| `DATASET_REPO` | `hyokwan/multi_modal_sample` | HF 데이터셋 레포 |
| `BASE_MODEL` | `google/gemma-3-4b-it` | 베이스 모델 ID |
| `OUTPUT_BASE_DIR` | `./models` | 학습 결과 저장 루트 경로 |
| `NUM_EPOCHS` | `5` | 학습 에폭 수 |
| `LEARNING_RATE` | `2e-4` | 학습률 |
| `MAX_SEQ_LENGTH` | `768` | 최대 시퀀스 길이 |
| `PER_DEVICE_TRAIN_BATCH_SIZE` | `1` | GPU당 배치 크기 |
| `GRAD_ACCUM` | `4` | Gradient Accumulation Steps |
| `LORA_R` | `16` | LoRA rank |
| `LORA_ALPHA` | `32` | LoRA 스케일링 계수 |
| `LORA_DROPOUT` | `0.05` | LoRA dropout 비율 |
| `LOGGING_STEPS` | `10` | TensorBoard 로그 주기 |
| `SAVE_STEPS` | `100` | 체크포인트 저장 주기 |

> `.env` 파일 값만 바꾸면 코드 수정 없이 하이퍼파라미터 시뮬레이션 가능

## 출력 경로 구조

```
{OUTPUT_BASE_DIR}/
└── gemma3_multimodal_lora_YYYYMMDD_HHMMSS/   ← OUTPUT_DIR
    ├── adapter_model.safetensors              ← LoRA 가중치
    ├── adapter_config.json
    ├── preprocessor_config.json
    ├── training_loss.png                      ← 손실 시각화
    └── tensorboard/                           ← TENSORBOARD_DIR
        └── events.out.tfevents.*
```

**TensorBoard 실행:**
```bash
tensorboard --logdir ./models/gemma3_multimodal_lora_YYYYMMDD_HHMMSS/tensorboard
```

## DB 저장 스키마 (`training_runs` 테이블)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| `id` | INT PK | 자동 증가 |
| `run_timestamp` | DATETIME | 학습 완료 시각 |
| `model_name` | VARCHAR(255) | 베이스 모델명 |
| `dataset_repo` | VARCHAR(255) | 데이터셋 레포 |
| `output_dir` | VARCHAR(512) | 어댑터 저장 경로 |
| `tensorboard_dir` | VARCHAR(512) | TensorBoard 경로 |
| `train_loss` | FLOAT | 최종 학습 손실 |
| `train_runtime_sec` | FLOAT | 학습 소요 시간(초) |
| `train_samples_per_sec` | FLOAT | 초당 처리 샘플 수 |
| `global_step` | INT | 전체 학습 스텝 수 |
| `num_epochs` / `learning_rate` / `lora_r` / `lora_alpha` / `lora_dropout` / `max_seq_length` / `grad_accum` / `batch_size` | 각 타입 | 학습 설정값 |

## .py 스크립트 구조 (`★02. 파인튜닝_멀티모달데이터_Gemma3.py`)

| 함수 | 역할 |
|------|------|
| `setKoreanFont()` | matplotlib 한글 폰트 자동 설정 |
| `loadConfig()` | `.env` 전체 파싱 → 설정 딕셔너리 반환 |
| `buildMessages(example)` | 샘플 → 모달리티별 메시지 구조 변환 |
| `getCollateFn(processor, model)` | processor/model 캡처한 `collateFn` 클로저 반환 |
| `plotTrainingLoss(trainer, outputDir)` | 학습 손실 PNG 저장 |
| `saveTrainingResultToDB(...)` | MySQL `training_runs` 자동 생성 + 저장 |
| `inferUnified(question, model, processor, ...)` | 텍스트/이미지 추론 |
| `main()` | 전체 학습 파이프라인 실행 |

**실행:** `python "★02. 파인튜닝_멀티모달데이터_Gemma3.py"`
